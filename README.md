# Storm-V2
Storm Selfbot improved in many ways.
I Still have big plans for this so please stay tuned!

# Instructions

1. ```git clone https://github.com/Shadyily/Storm-V2```
2. ```cd```
3. ```cd Storm-V2```
4. ```nano config.json```
5. Put your token in "config.json"
6. ```./start```
